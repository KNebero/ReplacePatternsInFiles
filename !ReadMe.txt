настройка config файла:
файл содержит следующие данные: путь до директории, где необходимо изменять файлы, количество потоков, в которые должна работать порграмма, перечисление шаблонов.

Путь до файла имеет следубщий формат:
path="path", где "path" - это ваш путь.
Наример, path=C:\dir

Количество потоков указывается в следующем формате:
threads='n', где 'n' - это чсило потоков.
Например, threads=3

Каждый из шаблонов указывается следующим образом:
pattern "pttrn"->"text", где pttrn - шаблон, text - его замена.
Например, pattern "<%Colors>"->"Black"
Обратите внимание, что шаблон и его замена обособляются двойными кавычками.

Каждая настройка указывается с начала новой строки, при этом указывать их можно в произвольном порядке.
